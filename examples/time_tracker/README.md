# Time Tracker Examples

This directory contains examples and demos for the Time Tracker multi-agent system.

## Running the Demo

### Interactive Demo

```bash
python e2e_time_tracker_demo.py
```

This will walk you through:
- Simulating a full workday of activities
- Generating various reports
- Viewing analytics and insights
- Detecting time gaps
- Exporting data

### Quick Demo

```bash
python e2e_time_tracker_demo.py --quick
```

Runs a non-interactive version showing all features.

## What the Demo Shows

1. **Activity Tracking** - Logging activities throughout the day
2. **Report Generation** - Daily reports, breakdowns, charts
3. **Analytics** - Productivity scores, focus metrics, top activities
4. **Time Wasters** - Identifying unproductive time
5. **Insights** - Personalized improvement suggestions
6. **Gap Detection** - Finding missing time entries
7. **Data Export** - JSON, CSV, and backup creation
